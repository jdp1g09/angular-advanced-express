Directives - Types of Scope

This <input> box is linked to the controller's parentVariable variable.

The three sections below are directives with the same content. They are constructed as children of the scope containing the parentVariable. They are all built with very similar HTML - an <input> field and a <p> node that displays parentVariable in a different form, depending on the directive's scope attribute

For a detailed description of how each of them work, use the 'left' and 'right' button to look through the slides (which are conveniently stored inside a re-usable carousel directive). They try to explain the 3 concepts with an analogy of a child going to a nightclub and the parent wanting to know all of the details of what is going on inside the club.
