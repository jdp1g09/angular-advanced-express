Below is an interactive example of 3 different directives, all blended into one.

  1. The table itself is the reusable-table HTML-only directive. This has an isolated scope, scope : {}, taking an input of table-data. It converts that array of row and column information and draws a table. There is no controller or link functionality here.
  2. The table cells use a HTML & functional reusable-inline-editor directive. This also has an isolated scope, scope : {}, taking an input of editing-value. It binds that value to a div and input node to create the editable cell. It has some logic stored in a controller, toggling the visibility of the div and input nodes, and also injects HTML by replacing the reusable-inline-editor tags in the source HTML.
  3. The inline editor then used a functionality-only directive called reusable-focus-me. This directive uses the link option in the directive. It watches the reusable-focus-me attribute for change to a true value. When it detects it, it give focus to the input element and hightlights it's content.
